展示 Apache Geode 作為分散式記憶體資料網格的功能，並整合 Spring Boot 微服務架構。

## 功能涵蓋

### 核心功能

| 功能 | 說明 |
|------|------|
| 基本 CRUD 操作 | Customer 和 Account 的增刪改查 |
| 帳戶操作 | 存款、提款、轉帳功能 |
| 故障轉移 | PARTITION_REDUNDANT Region |
| ACID 交易 | CacheTransactionManager |
| 持續查詢 (CQ) | 即時監控帳戶餘額變更 |
| 磁碟持久化 | PARTITION_REDUNDANT_PERSISTENT |
| WAN 複製 | 雙站點雙向資料複製 |

### 進階功能

- Function Execution：伺服器端聚合運算
- Security：認證授權機制
- Lucene 全文搜尋
- Cache Writer/Loader
- Async Event Queue
- Delta Propagation
- Expiration Policies
- Snappy Compression

## 監控整合

- Micrometer + Prometheus + Grafana
